# Computer Networks & Social Networks Preparation

## Overview

This repository contains my preparation notes and practice materials for **Computer Networks (CN)** and **Social Network Analysis (SNA)**.
The goal is to build a **strong conceptual understanding of networking fundamentals** required for **technical interviews, placements, and academic exams**.



## Learning Approach

* Focus on **conceptual understanding instead of memorization**
* Use **real-world examples** to understand networking behavior
* Practice **interview-oriented questions**
* Revise core concepts regularly


## Goals

* Strengthen **fundamentals for placements**
* Prepare for **technical interviews**
* Improve problem-solving in networking concepts
